Installation

Getting Started

Luna Signup Form Wizard, uses Jquery for javascript and Bootstrap for responsive styles. This theme also includes Icomoon Line Icons

First of all, extract "luna-full.zip" file to a folder, or your server via ftp. You will see 2 directory, Documentation and Luna, open Luna folder.

All css, js, html,php files, and images are in this folder. You can copy one of directory (general, payment, etc.. ) all this files to your working directory or server.

Customization

You can customize theme, changing some css codes. Background color, left side color, right side color etc.


Left SideBar color

Go to css/style.css file. Find .luna-signup-left-overlay and .luna-signup-left element. Change both background to your custom color. 

...
.luna-signup-left { ... background:#48A5EA; }
.luna-signup-left-overlay { ... background: .. #48A5EA; ... }
...

Main Background color

Go to css/style.css file. Find .luna-signup-right element. and change both background to your custom color. 

...
.luna-signup-right { ... background:#f55661; ... }
...

How to remove loader

To remove preloader effect very easy. Just open any HTML files and remove these lines below;

...
<div class="luna-loader-container">
    <div class="luna-loader"></div>
</div>
...

Form Wizard

Steps

This theme comes from a default wizard which has three steps and a confirm step. This steps are;

  • Account Information
  • Personal Information
  • Contact Information
  • Confirm Section

Each step has an html content in .luna-steps element

...
<div class="step step-active" data-step-id='1'>
    ...
</div>
...

.step-active class show us, active step.

data-step-id attribute is step id, starts with 1 and must be ordered number

Info and error messages

Info messages have help-block clases, and takes places after input elements. they are visible by default.
Error messages have comes from validation. Only show when validate form or input. 

...
<div class="step" data-step-id='4'>
...
<div class='help-block'>Please enter a valid 10 digit US phone number without country code</div>
...
</div>

Validation

This theme uses Jquery Validation plugin. https://jqueryvalidation.org/

Required Fields

Add "required" attribute to required fields


<input type="text" class="formInput" required id="name" name="name" >
And look at for advanced rules jquery plugin documentation.
https://jqueryvalidation.org/ 


var form = $("form[name='signupForm']");
        form.validate({
            rules : {
                password : {
                    minlength : 6
                },
                password_confirm : {
                    minlength : 6,
                    equalTo : "#password"
                }
            },
            ignore: ":hidden",
            errorPlacement: function (error, element) {
                
                var formGroup = element.parents(".form-group");
                formGroup.find(".errorIcon").remove();
                formGroup.append('<span class="errorIcon"><i class="icon icon-info"></i></span>');
                element.parents(".form-group").find(".errorIcon").show().find("i").attr("title", error.text()).tooltip({container: 'body'});
            }
        })

Add Custom Step

  1. Open index.html file, and find element which has class luna-steps
  2. You will see div elements has class step. Copy this last element, and paste belowe.
  3. Change data-step-id attribute, increase +1.
  4. Add fields to this step. Change input names
  5. Go to js/main.js file and add validation for this input. You can look at validation to add validation.
  6. Go to confirm section and a custom row for each input. There should be an html element has class equals to input names.
  7. That's it!

Submiting Form

There is a php file called smtp.php in main directory. This file send form data with SMTP Protocol. You should only change mail settings. 


$mail->Host = "mail.wmedya.com";
//Set the SMTP port number - likely to be 25, 465 or 587
$mail->Port = 587;
//Whether to use SMTP authentication
$mail->SMTPAuth = true;
//Username to use for SMTP authentication
$mail->Username = "info@wmedya.com";
//Password to use for SMTP authentication
$mail->Password = "password";
//Set who the message is to be sent from
$mail->setFrom('info@wmedya.com', 'Wmedya');
//Set an alternative reply-to address
$mail->addReplyTo('from@host.com', 'Wmedya');
//Set who the message is to be sent to
$mail->addAddress('whoto@host.com');
